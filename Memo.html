コマンド
用途 コマンド 引数
アプリ作成　 composer create-project laravel/laravel 末尾に [アプリ名]
アプリ初期設定　 composer install 引数 なし
サーバー起動　 php artisan serve 引数 なし
コントローラ作成　 php artisan make:controller 末尾に [ コントローラ名 ]
マイグレーション作成　 php artisan make:migration 末尾に [ create_xxxx_table ]
★マイグレーション自動実行　 source script/migrate.sh GitBashで実行する
マイグレーション
実行（上から順番通りに）　		 php artisan migrate:reset 全てのマイグレーションを元に戻す。
php artisan migrate --path=database/migrations/2025_05_29_025928_create_books_table.php
php artisan migrate --path=database/migrations/2025_05_29_040341_create_members_table.php
php artisan migrate
シーダ作成　 php artisan make:seeder 末尾に [ XxxxTableSeeder ] 先頭大文字
★シーダ実行実行　 source script/seeder.sh GitBashで実行する
シーダ実行　 php artisan db:seed --class="" 末尾に [ "XxxxTableSeeder" ] 先頭大文字
モデル作成　 php article make:model 末尾に [ Xxxx ] 先頭大文字
