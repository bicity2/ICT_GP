＜手動実行＞ 実行環境：GitBash / 実行場所：book / 必ず上から順に実行する
内容　　　　　　　 コマンド　　　　　　　　　　　
reset migration php artisan migrate:reset
migration（1/3） php artisan migrate --path=database/migrations/2025_05_29_025928_create_books_table.php
migration（2/3） php artisan migrate --path=database/migrations/2025_05_29_040341_create_members_table.php
migration（3/3） php artisan migrate
seeder(1/3) php artisan db:seed --class="BooksTableSeeder"
seeder(2/3) php artisan db:seed --class="MembersTableSeeder"
seeder(3/3) php artisan db:seed --class="ReviewsTableSeeder"


コマンド
用途 コマンド 引数
アプリ作成　 composer create-project laravel/laravel 末尾に [アプリ名]
アプリ初期設定　 composer install 引数 なし
サーバー起動　 php artisan serve 引数 なし
コントローラ作成　 php artisan make:controller 末尾に [ コントローラ名 ]
マイグレーション作成　 php artisan make:migration 末尾に [ create_xxxx_table ]
★マイグレーション自動実行　 source script/migrate.sh GitBashで実行する
